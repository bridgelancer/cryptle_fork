Welcome to Cryptle
==================

Welcome to Cryptle's documentation. Cryptle is a Python algorithmic trading
framework. Create your first strategy, backtest it, then go straight to
livetrading in :doc:`quickstart`. Once you are familiar with the basic usage of
Cryptle, we recommend you to proceed to the :doc:`Usage guides <guides>`, where
there would be more detailed information about individual components in Cryptle.

Installation
------------
Cryptle is not yet available on PyPI. Obtain a copy of the source distribution
from github to install it:

.. code-block:: bash

   git clone https://github.com/pinealan/cryptle.git
   cd cryptle

Then install locally with pip:

.. code-block:: bash

   pip install .
