Welcome to Cryptle
==================

Welcome to Cryptle's documentation. Cryptle is a Python algorithmic trading
framework. Get started with an overview at the 5 minute :doc:`quickstart`. Once
you are familiar with the basic concepts, we recommend you to proceed to the
:doc:`User guides <guides>`, where there would be detailed information about 
Cryptle.

Those who are familiar with the framework can head over to :doc:`advanced` to
find specific task-oriented HOWTO recipes and explanation for core concepts used
throughout the framework.
